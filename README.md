## 🚀 Projects

### 🌲 Forest Carbon Stock Estimation & Carbon Sequestration Mapping
- Built an end-to-end forest biomass estimation pipeline for Nainital District, Uttarakhand using 117,933 GEDI L4A footprints, Sentinel-2 spectral indices (NDVI, EVI, NDMI, NBR, SAVI), and SRTM terrain features across 18.6M forest pixels
- Trained XGBoost and Random Forest regressors on log-transformed AGBD targets; achieved R² = 0.65 (log-space), MAE = 56.3 t/ha, consistent with published GEDI biomass models over Himalayan terrain
- Identified systematic GEDI waveform quality degradation with elevation (R² = 0.63 below 700 m vs. 0.30 above 1200 m) across 22,408 held-out test footprints
- Generated wall-to-wall carbon maps using IPCC (2006) conversion factors: district total 36.95 MtC / 135.6 MtCO2e across 7,458 km² of forest, with a 4-zone sequestration potential classification layer
- Automated the full pipeline — GEE export → feature engineering → model training → raster prediction → interactive Folium map — using Rasterio, GeoPandas, XGBoost, and Scikit-learn
